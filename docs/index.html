About SambaLite

SambaLite is a lightweight, modern, and open-source Android client for SMB/CIFS shares (Samba). It provides a minimalistic, reliable, and secure tool for accessing SMB shares on local networks without unnecessary features, ads, or bloat.

Note: SambaLite is an independent open-source project and has no official affiliation with the Samba Project or SerNet. The name refers solely to the supported SMB/CIFS protocols.

Key Features

  • Connect to SMB/CIFS shares with username/password and domain
  • Browse files and folders
  • Download and upload files
  • Delete and rename files
  • Search with wildcards
  • Modern UI with Dark Mode support
  • Manage multiple connections
  • Secure credential storage
